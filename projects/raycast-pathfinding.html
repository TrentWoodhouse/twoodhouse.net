Project Information

Yes, this is an implementation of a pathfinding algorithm, but not your typical one. The grid here is just for show. I've freed up some constraints and allowed our pc to be in a "free environment", needing to use raycasting in order to mimic the approach of how a real life person would solve the problem of finding the target.

This project is currently unfinished. Currently, a majority of the view component has been implemented. A 32x18 grid is used for ease of drawing orderly walls amd placing the pc and target, but is not used for the given pathfinding algorithm, which is instead based off the 1600x900 grid that makes up the canvas.
